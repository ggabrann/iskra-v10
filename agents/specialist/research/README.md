# Specialist — исследование
Роль: поиск и структурирование источников.

## Контракт (v1.0)
- **Intents:** исследование
- **Inputs:** контекст, ограничения, предшествующие артефакты
- **Outputs:** `report.md` + артефакты в `runs/<run_id>/artifacts/`

## Первое задание (пример)
минимум 5 источников + сводка на 1–2 стр.
