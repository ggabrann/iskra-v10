# CONTRIBUTING to Iskra v10 (Minimal Team)

## Ветки
- Префиксы: `manifest/`, `feat/`, `fix/`, `docs/`, `ci/`, `routes/`.
- Одна тема — один PR.

## PR
- Ссылкуй Issue/ADR, если есть.
- Прогони базовые проверки: валидный JSON/YAML, наличие `routes/ritual.yaml`.
- Добавляй метки вида `role:specialist/<area>` (например, `role:specialist/research`).

## SLO перед merge
- Clarity ≥ 0.80
- Drift ≤ 0.30
- Для исследований: ≥ 2 источника в артефактах.
